JZedit

JZedit is a open source cloud editor and development enviroment (IDE) for websites, apps and Node.JS services.
You can run it locally as a "native" web app, or in the "cloud" (in a web browser).

When running in the cloud, the editor needs to connect to a server for features such as FTP and Source control management (Mercurial) to work.
Additionally, when using the cloud editor you get webhosting space to host web pages and Node.JS services ...

Want to make a web page or app ? And have it running live in the cloud five minuts from now ?

All you have to know is some JavaScript, HTML and CSS.

Deoply your homepage/app or node.js service in one click.

Features

Besides an IDE for JavaScript and NodeJS with function-list, auto-completion, auto-formatter and debugger, etc, the webIDE also has:

  • Static site generator with WYSIWYG mode
  • Mercurial source control management integration
  • SSH/SFTP and FTP/FTPS integration
