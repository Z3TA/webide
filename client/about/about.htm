What is webide.se ?

webide.se is JZedit running on a server.

webide.se is a cloud developer environment where you can compile static web sites and develop Node.JS micro-services.
webide.se also gives web hosting space for static web sites and Node.JS micro services, making it possible to both develop and run (host, in production) full stack JavaScript web applications from any device that have a web browser, without need for plugins or installing anything!

What is JZedit

JZedit is a editor for programming, specialized in JavaScript, CSS, HTML and web development, with IDE-like features such as auto complete and built in version control (Mercurial) support, and a built in Static site generator.

JZedit is open source and can run locally on a PC or any device supporting Node.JS.
It can also run on a headless private server where you connect to it from a browser.

Features

Grouped file tabs Automatic indentation management, save time not having to indent your code. Zoom out Alt+Z with markdown topics highlighted, makes it faster to navigate in large files. Smart auto-completion of JavaScript variables Tab. Static site generator with JavaScript support, kinda like classic ASP, but with support for Node.JS modules. WYSIYG and editor side by side, write HTML without knowing HTML. Connect to FTP/FTPS/SFTP and edit files remotely. Commit from within the editor and show annotation

Architecture

JZedit's source code is made of vanilla (no transpiling) JavaScript, HTML and CSS, and use HTML Canvas for rendering.
JZEdit's architecture is plugin and event driven, which makes it fun and easy to extend with new features. Back-end and front-end is separated in order to make everything work in the browser.
JZedit has a small core of only a few files. All functionality, even the rending, are implemented as plugins. Each plugin is independent - they do not depend on each other - avoiding complexity.
Plugins can be hot loaded, so while making a plugin, you can refresh it without reloading the editor.
