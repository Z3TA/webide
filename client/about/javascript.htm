JavaScript

JavaScript is dynamic programming language, which makes it fast and fun to program, and lets you quickly iterate and add new functionality.
It was designed for use in Browsers, for glue code in graphical user interfaces, but has in recent years become a very popular general purpose language, used for everything, even where it doesn't make sense.

This editor is specially made for programming JavaScript and Node.JS apps. And the editor itself is of course made (using this editor), in JavaScript and Node.JS.

The code is parsed at every key stroke, which allows for some of the following features:

  • Make sure your code is always properly indented
  • Context aware auto-complete.
  • A function list
  • Highlighting common mistakes
  • Show console.log() messages and errors inline (in the editor)
  • The parser also understands HTML, CSS, PHP, and Classic ASP, so you can mix those.

    • Meanwhile the editor is developed with speed and performance in mind. It uses incremental parsing to achieve less then 1ms latency between input and render.
    Which also allows it to run on low end devices. Most of the heavy lifting is done by the back-end server running in "the cloud". But you can also run the editor locally with both back-end and front-end on your own computer.

    In order to do this, the editor doesn't sacrifice speed and performance.
