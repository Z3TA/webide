Run the editor locally

How to run the editor as a native app on your computer.

The editor back-end/server is a Node.js script, while the front-end/client is a "progressive" web app which is loaded in a chrome-less browser window.

1. Install Node.JS if it's not already installed on your system

Download and install the (LTS) Node.js package for your operating system.

2. Download the latest release of JZedit (the editor) for your operating system.

If you are using Windows, click the link that has win-x64 in it, and the highest number. Example: https://www.webtigerteam.com/jzedit/download/jzedit-v1_alpha-3041-win-x64.zip
Replace 3041 with the highest number (latest release)

3. Extract the files

On Windows or a graphical user interface, right click on the file you downloaded and select "Extract files".
