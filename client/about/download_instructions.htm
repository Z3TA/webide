Run the editor locally

How to run the editor as a native app on your computer.

The editor back-end/server is a Node.js script, while the front-end/client is a progressive web app.

1. Install Node.JS if it's not already installed on your system

Download and install the Node.js package for your operating system.

2. Download the latest release of JZedit (the editor) for your operating system.
