Run the editor locally

How to download and run the editor as a native app on your computer.

1. Install Node.JS if it's not already installed on your system

Download and install the (LTS) Node.js package for your operating system.

2. Download the latest release of JZedit (the editor) for your operating system.

If you are using Windows, click the link that has windows in it, and the highest number. Example: https://www.webtigerteam.com/jzedit/download/jzedit-v1_alpha-3085-windows.zip
Replace 3085 with the highest number (latest release)

3. Extract the files

If you are on Windows, first right-click on the zip-file, and click "Properties". At the bottom, next to "Security: This file came from another computer" click "Unblock" and then "Apply".

Then right click on the file you downloaded and select "Extract files".

How to start

The editor back-end/server is a Node.js script, while the front-end/client is a "progressive" web app which is loaded in a chrome-less browser window.

Windows

Run the server and client by double clicking on "start.bat"

Create a link on the desktop by double clicking on "create_shortcut.vbs".
