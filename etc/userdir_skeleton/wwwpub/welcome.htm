Welcome to webide.se

This is a "integrated development environment" (IDE) that lets you develop And host Node.JS applications, including databases.
Create and edit web sites and apps. And also host them! For free!

You can for example create a Node.JS HTTP server, just replace the port-nr with /sock/nameofapp and the app will be accessible from http://nameofapp.%USERNAME%.%DOMAIN% or if you just want to make a static web site/app, place the files in the /wwwpub/ folder, and they can be accessed from http://%USERNAME%.%DOMAIN%/

Press F1 to see documents/manual, and feel free to send questions and feedback to editor@webtigerteam.com

Useful keyboard shortcuts are Ctrl+O to open files, Tab for auto-completions, and Ctrl+S to save. Or right click with the mouse to see more options.

To get started, try cloning an existing project: Right click, and pick "Clone a repository ..."

Happy coding!
