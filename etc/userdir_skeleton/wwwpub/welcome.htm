This is a web IDE/editor for creating Node.JS micro-services, web app's, web sites, blog's and blog posts, interanet web sites, web-shops, agar.io clones, etc ...

Lets get started

This file is placed in your wwwpub folder. All files placed in /wwwpub/ can be accessible from https://%USERNAME%.%DOMAIN%/

Try make some changes to this file, and save with Ctrl + S or "Save file" in the editor's right-click menu. And open the URL below in a new browser window or tab:
http://%USERNAME%.%DOMAIN%/welcome.htm

Normally you would have to refresh/reload F5 the browser tab/window to see changes after saving ...
But with the editors "Web Preview" function, you can see your changes live! Try right clicking and select "Web Preview" in the editors menu!

The "Web Preview" function also gives auto-complete on any JavaScript/library,
automatically updates the style if you save a CSS file,
and it also works with the static site generator (SSG)!
The SSG also supports "what you se is what you get" (WYSIWYG) editing!
