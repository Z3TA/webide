This is the source code of a web page (html file).

This file is placed in your wwwpub folder, all files splaced in /wwwpub/ will be accessable from http://%USERNAME%.webide.se/
Try opening the url http://%USERNAME%.webide.se/welcome.htm in another browser window or tab !

HTML is a friendly markup language that is easy to learn, but JZedit also comes with a built in "What you see is what you get" (WYSIWYG) editor.
Go back to the editor window/tab and Right click to bring up the menu, and then click on WYSIWYG. Any change you make in either the code window or the preview window will be visible in both.
Try to make some changes to this document and see what happends !

The changes will become live as soot as you hit Ctrl + S to save.
