Welcome to webide.se

You are looking at the source code of a web page (html file).

This file is placed in your wwwpub folder, and all files placed there will be accessable from http://%USERNAME%.webide.se/ (this is the address to your web site)

Try making some changes to this file, and save with Ctrl + S.
Then goto http://%USERNAME%.webide.se/welcome.html in another tab/window.

Static site Generator

To manage larger web sites, the editor has a Static Site Generator (SSG).
Show the SSG manager by hitting F9, or select "Static Site Generator" in the menu that comes up when you do a mouse right-click .

Using the SSG to write a new blog post

  1. In the SSG manager click New Page.
  2. Preview the page: Ctrl + F9 or Preview in SSG manager.
  3. Ctrl + S to save the file as /my_web_site/source/blog/blog_post.html
  4. Publish the site Ctrl + Alt + Shift + F9 or Publish in SSG manager.

Adding images to a web page

Drag a file from your computer to the editor, and select a path to save it as.

Running Node.JS scripts

Ctrl + P then type hello.js and press Enter.
Then press F1 to run the file/script.
