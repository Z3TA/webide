Static site Generator

To manage larger web sites, the editor has a Static Site Generator (SSG).
Show the SSG manager by hitting F9, or select "Static Site Generator" in the menu that comes up when you do a mouse right-click .

Using the SSG to write a new blog post

  1. In the SSG manager click New Page and Ctrl + S to save it in /my_web_site/source/blog/name_of_file.html. (make sure it's saved in the "blog" folder)
  2. Preview the page: Ctrl + F9 or Preview in SSG manager.
  3. Write the blog post and hit Ctrl + S to save when you are done.
  4. Publish the site Ctrl + Alt + Shift + F9 or Publish in SSG manager.

Adding images to a web page

Running Node.JS scripts
