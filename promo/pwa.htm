Progressive web apps

A progressive web app (PWA) is basically a web app that use new web technology, such as service workers, and "add to desktop" making it possible to "install" the web app on mobile phones and desktop computers.

Online IDE for PWA's

Webide.se is a PWA, for building PWA's and web sites.

It runs in the browser, no need to install anything. See screen-shots below:

Your own cloud IDE

You can download the cloud IDE and run it locally, or on your own server.

npm install -g jzedit

Open source

It's easy to make changes to the editor, add new plugins, or modify it to your needs.
It's built using vanilla HTML/CSS/JavaScript.

Native performance

JZedit is a new editor written from scratch using native web API's and the canvas element to render text.

Batteries included

It's not just an editor, it comes loaded with features like:

  • JavaScript intellisense
  • Semantic structure/coloring
  • Open new windows
  • Live preview
  • Colaborate editing
  • Inlines console.log messages
  • Version control
  • Static site generator
  • Terminal emulator
  • FTP/SFT support
  • Google drive

Host your PWA

Webide.se i gives you a free cloud shell, where you can also host your PWA, with database and Node.JS support!
