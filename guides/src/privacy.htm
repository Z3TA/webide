Privacy policy

This is the privacy policy for JZedit and webide.se

Personal information

There are no "telemetry" or tracking.

No personal information is collected, except IP address (see logging below).

Logging

The editor server has extensive logging capabilites, so only connect to a server that you trust!
Log files on webide.se are saved for one week.

File storage

Files on webide.se are stored on server hdd.
File storage is currently not encrypted.

Sharing & third party

We do not share any of your data. No third party will have access to your files or IP. Not even for statistics or analytics.

Only the server system administrator will be able to access your data.

Cookie policy

The editor use cookies and window.localStorage to store currently opened files, etc, so the editor can reopen them in the next session.

Editor login details are saved in locaStorage, so make sure you delete all browser history, including data, if you login to the server using a public computer.
