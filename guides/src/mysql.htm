Using a database

Creating a database

In the menu, click "Database manager", and then the button "Create new database".
You then get a dialog box where you type the name of the database.

Technical details

The editor uses the auth_socket mySQL plugin for authentication to the database, so you must use the mysql2 Node.JS module with authSwitchHandler specified in connection options.
With the auth_socket mySQL plugin you can login securely to the mySQL server without using a password. Only you can login using your username!
