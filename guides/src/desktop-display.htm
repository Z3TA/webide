Running GUI apps via local desktop/display

Activate your local desktop/display via via "View" > "Local desktop" in the window menu, or keyboard Ctrl+D.

Start apps via the terminal emulator: Right click and select "Terminal emulator", or via window menu "Tools" > "Terminal emulator".

If it's not working, wait 20 seconds, then try to activate the "local desktop"/display again.

Know issues

Windows might be gray/blank in certain Java apps such as the Android tools welcome screen.
