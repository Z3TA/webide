Editor customization

You can customize your experience by writing JavaScript or CSS that will be applied when the editor loads.
Right click and then left click on "Editor customization". Or create a new file and name it "jzedit_js_overload.js" or "jzedit_css_overload.css".
When saved, you will be asked to save the content locally, meaning it will be stored in your browser local storage.

If the editor runs locally on your machine, or you want to customize for all users, edit client/settings_overload.js
