Editor query string options

You can can change some editor settings using the query string. For example: https://webide.se/?embed=true&disable=menu,file_tabs

disable Disable menu,
editPage Open the source code for the URL. Example: editPage=https://zäta.com/en/about.htm will open the source for about.htm
start Example: start=terminal,
skiplogin Don't login
embed Example embed=true. Disables plugin's that doesn't want to run when embedded.
repo,clone Asks to clone a Git or Mercurial repository. Example: clone=https://github.com/Z3TA/jsql.git
