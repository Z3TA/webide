Collaborating with others

Using Mercurial, Git or other source control management (SCM) is probably the most common way to collaborate with others ...

In the cloud editor it's possible to invite others for real time collaboration, pair programming or code review!

  • Select "Invite collaborator" from the menu.

This will create a temporary password for logging into your account.
Then you can work with many people on the same document, and see each other type!
