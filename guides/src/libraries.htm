Working with third party JS libraries

The editor will give you auto-completion for any JavaScript library!

Put the library into a script-element, like this:

And put put your own code in "yourscript.js".
Then right click and select "Preview" in the editor menu.
The window will now auto-refresh every time you save your script, and the editor will be aware of all the variables and methods in the library script!

(Don't mind the body elements. Just leave them there or the editor will complain).

Non-minified version

Some libraries minify/uglify their releases so that variables and function names are replaced with single letter names, and that is not very useful, so you want to find the non-minified version of the library!
