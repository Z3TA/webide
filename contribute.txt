Contributing
============

Send any changes to zeta@zetafiles.org. Compress using zip/gzip. 

If your contribution is of any value I will implement it into the editor or publish it to the web.

If you want to be helpful, also use the Mercurial SCM so that it will be easier for me to manage:

1. Clone from http://hg.webtigerteam.com/js-editor/
2. Make sure it's up to date! (hg pull)
3. Make your changes
4. Commit the changes
5. Sending changes by email, with a note: "I give you full (copyright) permission to use this code"

Mercurial commands:
-------------------
# Clone the repository/folder
$ hg clone http://hg.webtigerteam.com/js-editor/ js-editor-new-feature

# Make changes and commit them...
$ hg ci -m "my changes"

# See latest changeset(s)
$ hg log 

$ hg export 3 > change3.diff 
$ hg export 4 > change4.diff 
# replace 3, 4, etc with the actual changeset


Importing
---------

Guide from: https://www.mercurial-scm.org/guide

# First clone the repository/folder to have an integration sandbox

$ hg clone project integration
$ cd integration
$ hg import change3.diff
$ hg import change4.diff

# Test changes ... If you like them, pull the changes into the main repository

$ cd ../project
$ hg pull ../integration
